== Travel Track

The Travel Track site provides a simple and extremely flexible way to journal about your travels.  Adding images, entries and exploring your destination are all easy to do with Travel Track.

== Architecture

* Hosted on Heroku

* Built on Ruby on Rails and jQuery

* Bootstrap CSS

* Icons from Font Awesome

* Fonts from Google Fonts

* Image hosting Amazon Web Services

* Image hosting with CarrierWave gem

* Google Map API

* Twitter Oauth
